### Install
```bash
npm i @fa-repo/shard-docs
```

### Dependencies

ShardDocs makes use of [React Router 4](https://reacttraining.com/react-router/web/guides/quick-start) for routing. Ensure that a router provider has been set further up the tree.

```jsx
import { BrowserRouter } from "react-router-dom";

render(
  <BrowserRouter>
    <ShardDocs title="Documentation title" />
  </BrowserRouter>,
  document.getElementById("root")
);
```
