```js
// example
const content = [
  { type: "document", name: "Install", document: [ <h1>Install</h1> ] },
  { type: "document", name: "Basic usage", document: [ <h1>Basic usage</h1> ] },
  {
    type: "category",
    name: "Examples",
    items: [
      { type: "document", name: "Use case A", document: [ <h1>Use case A</h1> ] },
      { type: "document", name: "Use case B", document: [ <h1>Use case B</h1> ] }
    ]
  },
  { type: "external-link", name: "Github", externalLink: "https://github.com" },
]
```

#### Document item

| Name       | Type   | Default | Required | Description                       |
|------------|--------|---------|----------|-----------------------------------|
| `type`     | string | `""`    | Required | Must be "document".                   |
| `name`    | string | `""`    | Required | Document name.                   |
| `document` | JSX    | `null`  |          | Document content.                 |

#### Category item

| Name       | Type   | Default | Required | Description                                       |
|------------|--------|---------|----------|---------------------------------------------------|
| `type`     | string | `""`    |          | Must be "category".                                     |
| `name`    | string | `""`    |          | Category name.                                     |
| `items`   | array  | `[]`    | Required | An array that accepts all items listed here.      |

#### External link item

| Name             | Type   | Default | Required | Description         |
|------------------|--------|---------|----------|---------------------|
| `type`           | string | `""`    | Required | Must be "external-link".          |
| `name`          | string | `""`    |          | Link text.          |
| `externalLink`   | string | `""`    | Required | Link path.          |
