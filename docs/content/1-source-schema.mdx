```js
// example
const content = [
  { type: "document", title: "Install", document: [ <h1>Install</h1> ] },
  { type: "document", title: "Basic usage", document: [ <h1>Basic usage</h1> ] },
  {
    type: "folder",
    title: "Examples",
    folder: [
      { type: "document", title: "Use case A", document: [ <h1>Use case A</h1> ] },
      { type: "document", title: "Use case B", document: [ <h1>Use case B</h1> ] }
    ]
  },
  { type: "external-link", title: "Github", externalLink: "https://github.com" },
]
```

#### Document item

| Name       | Type   | Default | Required | Description                       |
|------------|--------|---------|----------|-----------------------------------|
| `type`     | string | `""`    | Required | Must be "document".                   |
| `title`    | string | `""`    | Required | Document title.                   |
| `document` | JSX    | `null`  |          | Document content.                 |

#### Folder item

| Name       | Type   | Default | Required | Description                                       |
|------------|--------|---------|----------|---------------------------------------------------|
| `type`     | string | `""`    |          | Must be "folder".                                     |
| `title`    | string | `""`    |          | Folder title.                                     |
| `folder`   | array  | `[]`    | Required | An array that accepts all items listed here.      |

#### External link item

| Name             | Type   | Default | Required | Description         |
|------------------|--------|---------|----------|---------------------|
| `type`           | string | `""`    | Required | Must be "external-link".          |
| `title`          | string | `""`    |          | Link text.          |
| `externalLink`   | string | `""`    | Required | Link path.          |
