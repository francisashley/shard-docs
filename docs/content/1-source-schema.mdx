```js
// example
const content = [
  { name: "Install", content: <h1>Install</h1> },
  { name: "Basic usage", content: <h1>Basic usage</h1> },
  {
    name: "Examples",
    content: [
      { name: "Use case A", content: <h1>Use case A</h1> },
      { name: "Use case B", content: <h1>Use case B</h1> }
    ]
  },
  { name: "Github", content: "https://github.com" },
]
```

#### Page item

| Name        | Type          | Description                       |
|-------------|---------------|-----------------------------------|
| `name`      | `string`      | Page name.                        |
| `content`   | `JSX`|`null`  | Page content.                     |

#### Category item

| Name       | Type   | Description                                 |
|------------|--------|---------------------------------------------|
| `name`     | string | Category name.                              |
| `content`  | array  | An array containing items listed here.      |

#### Link item

| Name          | Type    | Description                                      |
|---------------|---------|--------------------------------------------------|
| `name`        | string  | Link name.                                       |
| `content`     | string  | Accepts internal and external links. E.g. /path/to/path. http://www.website.com, https://www.website.com |
