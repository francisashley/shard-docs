| Name                      | Type                      | Default   | Description                                                     |
|---------------------------|---------------------------|-----------|-----------------------------------------------------------------|
| `title`                   | string                    | `""`      | A title for your documentation. Appears at the top of the sidebar. |
| `basePath`                | string                    | `"/docs"` | A path prepended to every page.                         |
| `content`                 | array                     | `[]`      | Documentation data organised as a tree. See below for help.   |
| `hideBuiltWithShardDocs`  | boolean                   | `false`   | Hide "Built with @fa-repo/shard-docs" on sidebar. |
| `routerType`              | "hash"(defauot)/"browser" | `false`   | Use [BrowserRouter](https://reacttraining.com/react-router/web/api/BrowserRouter) or [HashRouter](https://reacttraining.com/react-router/web/api/HashRouter) for routing. |
